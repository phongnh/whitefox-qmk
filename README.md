## Setup

```
$ brew install qmk/qmk/qmk
$ git clone https://github.com/qmk/qmk_firmware.git ~/src/qmk_firmware
$ cd ~/src/qmk_firmware
$ util/qmk_install.sh
```

## Compile

```
$ git clone https://github.com/phongnh/whitefox-qmk.git ~/src/qmk_firmware/keyboards/whitefox/keymaps/phongnh
$ cd ~/src/qmk_firmware
$ make whitefox:phongnh
$ make whitefox:phongnh:flash
```
